## Installation 
The user should clone the repository and install the node express and sequelize packages. To connect to the database run `mysql -u root -p` and enter password from .env file. Then source the schema.sql. To seed the file run `npm run seed`. Finally to connect to the server run `npm start`. 

## Usage 
This application will allow users to view, add, edit, and delete categories, products, and tags.
